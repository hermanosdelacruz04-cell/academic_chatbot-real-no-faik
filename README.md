# Academic Chatbot AI (SQL + NLP) 🤖📚

Un chatbot académico de alta fidelidad diseñado para instituciones educativas, que utiliza una arquitectura híbrida de base de datos **PostgreSQL (Supabase)** e Inteligencia Artificial generativa (**Puter.js**).

## 🌟 Características Principales
- **Arquitectura Híbrida**: Búsqueda semántica en base de datos SQL para respuestas oficiales y respaldo de IA para contexto general basado en archivos PDF/TXT.
- **Gestión de Roles (RBAC)**: 
  - **Administrador**: Gestión total de usuarios y personal institucional.
  - **Docente**: Herramientas para alimentar el conocimiento del bot.
  - **Coordinador**: Supervisión de actualizaciones y estado del sistema.
  - **Estudiante**: Interfaz de chat optimizada con registro automático.
- **Persistencia**: Historial de chat vinculado a la cuenta del estudiante en la nube.
- **Diseño Premium**: Interfaz oscura estéticamente agradable con animaciones fluidas y mascota interactiva.

## 🛠️ Tecnologías Usadas
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Base de Datos**: Supabase (PostgreSQL).
- **Motor de IA**: Puter.js SDK (Modelos GPT).
- **Iconografía**: Phosphor Icons.

## 🚀 Instalación Rápida
1. Clona este repositorio o descarga los archivos.
2. Abre `script.js` y configura tus credenciales de **Supabase**.
3. Ejecuta el script SQL incluido en el Editor SQL de Supabase para crear las 9 tablas necesarias.
4. ¡Inicia el proyecto usando un servidor local (como Live Server de VS Code) y listo!
